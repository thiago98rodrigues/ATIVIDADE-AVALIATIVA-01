# ATIVIDADE-AVALIATIVA-01
Programação para Dispositivos Móveis I
React - Componentes, Estados e Formulários

## Objetivo
Desenvolver uma aplicação React utilizando os conceitos estudados até o momento:
* Componentes React;
* Props;
* Class Components;
* State;
* Formulários controlados;
* Select;
* Manipulação de múltiplos campos;
* Eventos de submissão.

## Situação-Problema
Uma clínica médica deseja cadastrar pacientes para uma campanha de prevenção à saúde.
Você deverá desenvolver uma aplicação React chamada: **Cadastro de Pacientes**
